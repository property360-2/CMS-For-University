Markdown Cheat Sheet

Learn how to format your content using Markdown syntax

Headings

# Heading 1

Largest heading

## Heading 2
### Heading 3
#### Heading 4

Text Formatting

**Bold text**

Makes text bold

*Italic text*

Makes text italic

~~Strikethrough~~

Strikes through text

`inline code`

For code snippets

Lists

Unordered List:

- Item 1
- Item 2
  - Nested item

Ordered List:

1. First item
2. Second item
3. Third item

Links & Images

[Link text](https://example.com)

Create hyperlinks

![Alt text](image-url.jpg)

Embed images

Advanced Syntax

Code Blocks

```python
def hello_world():
    print("Hello, World!")
```

Use triple backticks with language name for syntax highlighting

Blockquotes

> This is a blockquote
> It can span multiple lines

Use > for quotes

Horizontal Rules

---

Creates a horizontal line separator

Tables

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

Create tables with pipes and dashes
